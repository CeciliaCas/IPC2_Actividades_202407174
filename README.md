# 🚀 IPC2 - Actividades

¡Bienvenido(a)! 👋

Este repositorio contiene las actividades, prácticas y proyectos desarrollados durante el curso de **Introducción a la Programación y Computación 2 (IPC2)**.

## 📚 Contenido del Repositorio

A lo largo del curso se irán almacenando:

* 📝 Reportes de investigación y análisis.
* 💻 Proyectos desarrollados en C# y .NET.
* 🌳 Implementaciones de estructuras de datos.
* 🏗️ Ejercicios de arquitectura de software y patrones de diseño.
* 🔍 Prácticas de programación orientada a objetos.
* 📂 Actividades de laboratorio y documentación técnica.

## 🎯 Objetivo

Mantener organizadas todas las evidencias y entregables del curso, aplicando buenas prácticas de desarrollo de software y documentación.

* ASP.NET Core MVC
* Git
* GitHub
* Visual Studio Code
